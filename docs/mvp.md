# Cashtrack MVP Scope

The **Minimum Viable Product (MVP)** version of **Cashtrack** will be developed to validate the concept of the application and allow early users to test its essential features. The focus is to deliver a functional system with the minimum necessary to meet the core needs of users in personal financial management.

### 2. **Transaction Management**
- Recording of income and expenses  
- Categorization of transactions (e.g., food, housing, entertainment)  
- Ability to edit and delete transactions  

### 3. **Financial Dashboard**
- Display of current balance based on recorded transactions  
- Basic charts for expense analysis  
- Transaction history organized by date  

### 4. **User Settings**
- Update of personal information (name, email, password)  
- Display and notification preferences
